# Interface commentateur (mobile)

Étapes tactiles en cascade :
1. Saisie code match (5 caractères).
2. Sélection titulaires et remplaçants (listes simples, gros boutons).
3. Vérification numéros de maillot.
4. Choix formation (4-4-2, 4-3-3, 3-5-2...).
5. Affectation des postes.

Chaque étape valide automatiquement vers l'étape suivante pour minimiser les interactions.
